# CS:GO Major Draft Simulator

一款 CS:GO Major 模拟选秀游戏，灵感来自 [82-0.com](https://www.82-0.com/)。

## 在线游玩

👉 https://YOUR_USERNAME.github.io/csgo-major-draft

## 玩法

- 点击 **开始抽卡**，老虎机随机抽取地区和时代
- 从 **6 名备选选手** 中选择一位
- 点击位置按钮将其放置到阵容（指挥/狙击手/明星选手/突破手/绕后手）
- 重复 5 轮填满阵容
- 点击 **模拟 Major** 看结果

## 策略

- 选手放到**擅长位置** → 100% 评分
- 放错位 → 评分 -15%
- 早期时代选手有**时代惩罚**
- 每局一次**跳过**机会

## 数据库

覆盖 6 大地区 × 5 个时代 × 200+ 真实选手，从 2000s f0rest/NEO 到 2023-2025 donk/m0NESY。
